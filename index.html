tentangtentang diri saya

Sekilas tentang latar belakang saya.

Saya Gideon, anak keempat dari empat bersaudara. Saya lahir pada tanggal 17 September 2006 di sebuah desa kecil di Minahasa Tenggara yang disebut Kuyanga. Ayah saya adalah seorang petani, dan ibu saya adalah seorang guru. Ayah saya sangat mencintai musik. Sejak kecil, saya diajari cara bermain alat musik seperti gitar dan piano, dan ini menjadi hobi saya sampai sekarang. Hobi ini kemudian mebawa saya menjadi seorang pemain musik di gereja.


Musik memang menjadi hobi saya, tapi di satu sisi saya juga tertaik dengan Teknologi. Sejak saya mengenal laptop di usi Sekolah Dasar, saya mulai belajar bagaimana cara mengetik menggunakan Word, membuat laporan menggunakan Excel, dan membuat prsentasi menggunakan Power Point. Seiring berjalan waktu saya mulai tertarik dengan pemrograman sehingga setelah lulus SMA, saya memutuskan untuk melanjutan pendidikan di Politeknik Negeri Manado Program Studi Teknik Informatika.

Kemampuan yang saya miliki di bidang Desain dan Musik

Saat ini, saya sedang mengeksplorasi dunia HTML dan CSS di semester kedua saya sebagai mahasiswa. Meskipun masih dalam proses belajar, saya merasa senang bisa membuat halaman web sederhana yang terlihat rapi dan menarik. Belajar HTML dan CSS memberi saya pemahaman yang lebih dalam tentang bagaimana desain dan konten saling berinteraksi untuk menciptakan pengalaman pengguna yang baik.

Bermain piano dan gitar telah menjadi bagian tak terpisahkan dari hidup saya sejak masa kecil. Setiap alat musik ini memberikan pengalaman yang unik dan mendalam dalam ekspresi diri melalui musik. Ketika saya duduk di depan piano, jari-jari saya menari di atas tuts, menciptakan melodi yang penuh emosi dan keindahan. Gitar membawa saya ke dunia ritme yang menggetarkan jiwa, dengan tiap senarnya menghasilkan harmoni yang begitu menggugah. Melalui kedua alat musik ini, saya belajar banyak tentang kreativitas, ketekunan, dan ekspresi diri.
